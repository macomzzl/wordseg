# wordseg

### 简介                                                                                                       
基于词典的最短路径分词算法                                                                                     
                                                                                                               
### 运行方法                                                                                                   
1. 运行 python Main.py.                                                                                        
2. 在控制台可以看到在测试集上的准确率、召回率和F值.                                                            
3. 在目录下会生成eval.html文件，可以对比分词结果与标准答案，其中红色的为分词错误的结果，黑色的为分析正确的结果.
